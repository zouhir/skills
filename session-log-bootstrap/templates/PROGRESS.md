# Progress Log

Append-only log of completed tasks. The agent writes here after finishing each item from `TODO.md`. Newest entries go at the bottom. Do not edit past entries — corrections go in a new entry.

Format for each entry:

```
## YYYY-MM-DD HH:MM — <task title from TODO.md>

**Done:** What was actually built/changed, in one or two concrete sentences.

**Files:** `path/one.ts`, `path/two.ts`

**Decisions:** Non-obvious choices a future session needs to know. Omit if none.

**Blockers:** Anything incomplete or waiting on the user. Omit if none.

**Next up:** Specific follow-up if not the next TODO item, otherwise "next TODO item".

---
```
